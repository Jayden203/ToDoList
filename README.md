# ToDoList

A Django-based ToDoList web application for managing tasks, categories, and progress tracking.

## Features

- User registration, login, and logout
- Create, edit, delete, and update task status
- Category management
- Search, filter, and sort tasks
- Analytics dashboard with task summary and category distribution
- Responsive Bootstrap-based interface
- Basic accessibility improvements

## Tech Stack

- Python
- Django
- Bootstrap 5
- SQLite
